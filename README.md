# MoodSense App

MoodSense is a personalized mobile wellness application that tracks mood, sleep, and stress using AI-powered insights.

## Tech Stack

- React Native (Expo)
- Node.js (Backend)
- PostgreSQL (Database)
- Groq API (LLaMA 3 AI)

## Features

- User authentication
- Daily check-ins
- Mood history tracking
- AI chatbot and insights
- Notifications and resources
